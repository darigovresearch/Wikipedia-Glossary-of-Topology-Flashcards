"<a href=""https://en.wikipedia.org/wiki/Alexandrov_topology"" title=""Alexandrov topology"">Alexandrov topology</a> (Wikipedia Glossary of Topology)"¬"
The topology of a space <i>X</i> is an <a href=""https://en.wikipedia.org/wiki/Alexandrov_topology"" title=""Alexandrov topology"">Alexandrov topology</a> (or is <b>finitely generated</b>) if arbitrary intersections of open sets in <i>X</i> are open, or equivalently, if arbitrary unions of closed sets are closed, or, again equivalently, if the open sets are the <a href=""https://en.wikipedia.org/wiki/Upper_set"" title=""Upper set"">upper sets</a> of a <a href=""https://en.wikipedia.org/wiki/Poset"" class=""mw-redirect"" title=""Poset"">poset</a>.<sup id=""cite_ref-1"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-1"">&#91;1&#93;</a></sup>"
"Almost discrete (Wikipedia Glossary of Topology)"¬"
A space is almost discrete if every open set is closed (hence clopen). The almost discrete spaces are precisely the finitely generated zero-dimensional spaces."
"<a href=""https://en.wikipedia.org/wiki/Approach_space"" title=""Approach space"">Approach space</a> (Wikipedia Glossary of Topology)"¬"
An <a href=""https://en.wikipedia.org/wiki/Approach_space"" title=""Approach space"">approach space</a> is a generalization of metric space based on point-to-set distances, instead of point-to-point."
"Baire space (Wikipedia Glossary of Topology)"¬"
This has two distinct common meanings:
<ol align=""left"">
<li>A space is a <b>Baire space</b> if the intersection of any <a href=""https://en.wikipedia.org/wiki/Countable"" class=""mw-redirect"" title=""Countable"">countable</a> collection of dense open sets is dense; see <a href=""https://en.wikipedia.org/wiki/Baire_space"" title=""Baire space"">Baire space</a>.</li>
<li><b>Baire space</b> is the set of all functions from the natural numbers to the natural numbers, with the topology of pointwise convergence; see <a href=""https://en.wikipedia.org/wiki/Baire_space_(set_theory)"" title=""Baire space (set theory)"">Baire space (set theory)</a>.</li>
</ol>

"<a href=""https://en.wikipedia.org/wiki/Bounded_set"" title=""Bounded set"">Bounded</a> (Wikipedia Glossary of Topology)"¬"
A set in a metric space is <a href=""https://en.wikipedia.org/wiki/Bounded_set"" title=""Bounded set"">bounded</a> if it has <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> diameter. Equivalently, a set is bounded if it is contained in some open ball of finite radius. A <a href=""https://en.wikipedia.org/wiki/Function_(mathematics)"" title=""Function (mathematics)"">function</a> taking values in a metric space is <a href=""https://en.wikipedia.org/wiki/Bounded_function"" title=""Bounded function"">bounded</a> if its <a href=""https://en.wikipedia.org/wiki/Image_(functions)"" class=""mw-redirect"" title=""Image (functions)"">image</a> is a bounded set."

"<a href=""https://en.wikipedia.org/wiki/Cauchy_sequence"" title=""Cauchy sequence"">Cauchy sequence</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Sequence"" title=""Sequence"">sequence</a> {<i>x</i><sub><i>n</i></sub>} in a metric space (<i>M</i>, <i>d</i>) is a <a href=""https://en.wikipedia.org/wiki/Cauchy_sequence"" title=""Cauchy sequence"">Cauchy sequence</a> if, for every <a href=""https://en.wikipedia.org/wiki/Positive_number"" class=""mw-redirect"" title=""Positive number"">positive</a> <a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">real number</a> <i>r</i>, there is an <a href=""https://en.wikipedia.org/wiki/Integer"" title=""Integer"">integer</a> <i>N</i> such that for all integers <i>m</i>, <i>n</i> &gt; <i>N</i>, we have <i>d</i>(<i>x</i><sub><i>m</i></sub>, <i>x</i><sub><i>n</i></sub>) &lt; <i>r</i>."
"<a href=""https://en.wikipedia.org/wiki/Clopen_set"" title=""Clopen set"">Clopen set</a> (Wikipedia Glossary of Topology)"¬"
A set is <a href=""https://en.wikipedia.org/wiki/Clopen_set"" title=""Clopen set"">clopen</a> if it is both open and closed."
"Closed ball (Wikipedia Glossary of Topology)"¬"
If (<i>M</i>, <i>d</i>) is a <a href=""https://en.wikipedia.org/wiki/Metric_space"" title=""Metric space"">metric space</a>, a closed ball is a set of the form <i>D</i>(<i>x</i>; <i>r</i>)&#160;:= {<i>y</i> in <i>M</i>&#160;: <i>d</i>(<i>x</i>, <i>y</i>) ≤ <i>r</i>}, where <i>x</i> is in <i>M</i> and <i>r</i> is a <a href=""https://en.wikipedia.org/wiki/Positive_number"" class=""mw-redirect"" title=""Positive number"">positive</a> <a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">real number</a>, the <b>radius</b> of the ball. A closed ball of radius <i>r</i> is a <b>closed <i>r</i>-ball</b>. Every closed ball is a closed set in the topology induced on <i>M</i> by <i>d</i>. Note that the closed ball <i>D</i>(<i>x</i>; <i>r</i>) might not be equal to the <a href=""https://en.wikipedia.org/wiki/Closure_(topology)"" title=""Closure (topology)"">closure</a> of the open ball <i>B</i>(<i>x</i>; <i>r</i>)."
"<a href=""https://en.wikipedia.org/wiki/Closed_set"" title=""Closed set"">Closed set</a> (Wikipedia Glossary of Topology)"¬"
A set is <a href=""https://en.wikipedia.org/wiki/Closed_set"" title=""Closed set"">closed</a> if its complement is a member of the topology."
"<a href=""https://en.wikipedia.org/wiki/Closed_function"" class=""mw-redirect"" title=""Closed function"">Closed function</a> (Wikipedia Glossary of Topology)"¬"
A function from one space to another is closed if the <a href=""https://en.wikipedia.org/wiki/Image_(mathematics)"" title=""Image (mathematics)"">image</a> of every closed set is closed."
"<a href=""https://en.wikipedia.org/wiki/Closure_(topology)"" title=""Closure (topology)"">Closure</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Closure_(topology)"" title=""Closure (topology)"">closure</a> of a set is the smallest closed set containing the original set. It is equal to the intersection of all closed sets which contain it. An element of the closure of a set <i>S</i> is a <b>point of closure</b> of <i>S</i>."

"<a href=""https://en.wikipedia.org/wiki/Coarser_topology"" class=""mw-redirect"" title=""Coarser topology"">Coarser topology</a> (Wikipedia Glossary of Topology)"¬"
If <i>X</i> is a set, and if <i>T</i><sub>1</sub> and <i>T</i><sub>2</sub> are topologies on <i>X</i>, then <i>T</i><sub>1</sub> is <a href=""https://en.wikipedia.org/wiki/Coarser_topology"" class=""mw-redirect"" title=""Coarser topology"">coarser</a> (or <b>smaller</b>, <b>weaker</b>) than <i>T</i><sub>2</sub> if <i>T</i><sub>1</sub> is contained in <i>T</i><sub>2</sub>. Beware, some authors, especially <a href=""https://en.wikipedia.org/wiki/Mathematical_analysis"" title=""Mathematical analysis"">analysts</a>, use the term <b>stronger</b>."
"Comeagre (Wikipedia Glossary of Topology)"¬"
A subset <i>A</i> of a space <i>X</i> is <b>comeagre</b> (<b>comeager</b>) if its <a href=""https://en.wikipedia.org/wiki/Complement_(set_theory)"" title=""Complement (set theory)"">complement</a> <i>X</i>\<i>A</i> is <a href=""https://en.wikipedia.org/wiki/Meagre_set"" title=""Meagre set"">meagre</a>. Also called <b>residual</b>."
"<a href=""https://en.wikipedia.org/wiki/Compact_space"" title=""Compact space"">Compact</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Compact_space"" title=""Compact space"">compact</a> if every open cover has a <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> subcover. Every compact space is Lindelöf and paracompact. Therefore, every compact <a href=""https://en.wikipedia.org/wiki/Hausdorff_space"" title=""Hausdorff space"">Hausdorff space</a> is normal. See also <b>quasicompact</b>."
"<a href=""https://en.wikipedia.org/wiki/Compact-open_topology"" title=""Compact-open topology"">Compact-open topology</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Compact-open_topology"" title=""Compact-open topology"">compact-open topology</a> on the set <i>C</i>(<i>X</i>, <i>Y</i>) of all continuous maps between two spaces <i>X</i> and <i>Y</i> is defined as follows: given a compact subset <i>K</i> of <i>X</i> and an open subset <i>U</i> of <i>Y</i>, let <i>V</i>(<i>K</i>, <i>U</i>) denote the set of all maps <i>f</i> in <i>C</i>(<i>X</i>, <i>Y</i>) such that <i>f</i>(<i>K</i>) is contained in <i>U</i>. Then the collection of all such <i>V</i>(<i>K</i>, <i>U</i>) is a subbase for the compact-open topology."
"<a href=""https://en.wikipedia.org/wiki/Complete_space"" class=""mw-redirect"" title=""Complete space"">Complete</a> (Wikipedia Glossary of Topology)"¬"
A metric space is <a href=""https://en.wikipedia.org/wiki/Complete_space"" class=""mw-redirect"" title=""Complete space"">complete</a> if every Cauchy sequence converges."
"Completely metrizable/completely metrisable (Wikipedia Glossary of Topology)"¬"
See <b><a href=""https://en.wikipedia.org/wiki/Complete_space"" class=""mw-redirect"" title=""Complete space"">complete space</a></b>."
"Completely normal (Wikipedia Glossary of Topology)"¬"
A space is completely normal if any two separated sets have <a href=""https://en.wikipedia.org/wiki/Disjoint_sets"" title=""Disjoint sets"">disjoint</a> neighbourhoods."
"Completely normal Hausdorff (Wikipedia Glossary of Topology)"¬"
A completely normal Hausdorff space (or <a href=""https://en.wikipedia.org/wiki/T5_space"" class=""mw-redirect"" title=""T5 space""><b>T<sub>5</sub></b> space</a>) is a completely normal T<sub>1</sub> space. (A completely normal space is Hausdorff <a href=""https://en.wikipedia.org/wiki/If_and_only_if"" title=""If and only if"">if and only if</a> it is T<sub>1</sub>, so the terminology is <a href=""https://en.wikipedia.org/wiki/Consistent"" class=""mw-redirect"" title=""Consistent"">consistent</a>.) Every completely normal Hausdorff space is normal Hausdorff."
"<a href=""https://en.wikipedia.org/wiki/Completely_regular_space"" class=""mw-redirect"" title=""Completely regular space"">Completely regular</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Completely_regular_space"" class=""mw-redirect"" title=""Completely regular space"">completely regular</a> if, whenever <i>C</i> is a closed set and <i>x</i> is a point not in <i>C</i>, then <i>C</i> and {<i>x</i>} are functionally separated."
"<a href=""https://en.wikipedia.org/wiki/Completely_T3_space"" class=""mw-redirect"" title=""Completely T3 space"">Completely T<sub>3</sub></a> (Wikipedia Glossary of Topology)"¬"
See <b><a href=""https://en.wikipedia.org/wiki/Tychonoff_space"" title=""Tychonoff space"">Tychonoff</a></b>."
"Component (Wikipedia Glossary of Topology)"¬"

"<a href=""https://en.wikipedia.org/wiki/Connected_(topology)"" class=""mw-redirect"" title=""Connected (topology)"">Connected</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Connected_(topology)"" class=""mw-redirect"" title=""Connected (topology)"">connected</a> if it is not the union of a pair of <a href=""https://en.wikipedia.org/wiki/Disjoint_sets"" title=""Disjoint sets"">disjoint</a> nonempty open sets. Equivalently, a space is connected if the only clopen sets are the whole space and the empty set."
"<a href=""https://en.wikipedia.org/wiki/Connected_space"" title=""Connected space"">Connected component</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Connected_space"" title=""Connected space"">connected component</a> of a space is a <a href=""https://en.wikipedia.org/wiki/Maximal_set"" title=""Maximal set"">maximal</a> nonempty connected subspace. Each connected component is closed, and the set of connected components of a space is a <a href=""https://en.wikipedia.org/wiki/Partition_of_a_set"" title=""Partition of a set"">partition</a> of that space."
"<a href=""https://en.wikipedia.org/wiki/Continuity_(topology)"" class=""mw-redirect"" title=""Continuity (topology)"">Continuous</a> (Wikipedia Glossary of Topology)"¬"
A function from one space to another is <a href=""https://en.wikipedia.org/wiki/Continuity_(topology)"" class=""mw-redirect"" title=""Continuity (topology)"">continuous</a> if the <a href=""https://en.wikipedia.org/wiki/Preimage"" class=""mw-redirect"" title=""Preimage"">preimage</a> of every open set is open."
"Continuum (Wikipedia Glossary of Topology)"¬"
A space is called a continuum if it a compact, connected Hausdorff space."
"<a href=""https://en.wikipedia.org/wiki/Contractible_space"" title=""Contractible space"">Contractible</a> (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> is contractible if the <a href=""https://en.wikipedia.org/wiki/Identity_function"" title=""Identity function"">identity map</a> on <i>X</i> is homotopic to a constant map. Every contractible space is simply connected."
"Coproduct topology (Wikipedia Glossary of Topology)"¬"
If {<i>X</i><sub><i>i</i></sub>} is a collection of spaces and <i>X</i> is the (set-theoretic) <a href=""https://en.wikipedia.org/wiki/Disjoint_union"" title=""Disjoint union"">disjoint union</a> of {<i>X</i><sub><i>i</i></sub>}, then the coproduct topology (or <b>disjoint union topology</b>, <b>topological sum</b> of the <i>X</i><sub><i>i</i></sub>) on <i>X</i> is the finest topology for which all the injection maps are continuous."
"<a href=""https://en.wikipedia.org/wiki/Cosmic_space"" title=""Cosmic space"">Cosmic space</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Continuous_function"" title=""Continuous function"">continuous</a> <a href=""https://en.wikipedia.org/wiki/Image_(mathematics)"" title=""Image (mathematics)"">image</a> of some <a href=""https://en.wikipedia.org/wiki/Separable_space"" title=""Separable space"">separable</a> <a href=""https://en.wikipedia.org/wiki/Metric_space"" title=""Metric space"">metric space</a>.<sup id=""cite_ref-2"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-2"">&#91;2&#93;</a></sup>"
"<a href=""https://en.wikipedia.org/wiki/Countable_chain_condition"" title=""Countable chain condition"">Countable chain condition</a> (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> satisfies the countable chain condition if every family of non-empty, pairswise disjoint open sets is countable."
"Countably compact (Wikipedia Glossary of Topology)"¬"
A space is countably compact if every <a href=""https://en.wikipedia.org/wiki/Countable"" class=""mw-redirect"" title=""Countable"">countable</a> open cover has a <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> subcover. Every countably compact space is pseudocompact and weakly countably compact."
"Countably locally finite (Wikipedia Glossary of Topology)"¬"
A collection of subsets of a space <i>X</i> is <b>countably locally finite</b> (or <b>σ-locally finite</b>) if it is the union of a <a href=""https://en.wikipedia.org/wiki/Countable"" class=""mw-redirect"" title=""Countable"">countable</a> collection of locally finite collections of subsets of <i>X</i>."
"<a href=""https://en.wikipedia.org/wiki/Cover_(topology)"" title=""Cover (topology)"">Cover</a> (Wikipedia Glossary of Topology)"¬"
A collection of subsets of a space is a cover (or <b>covering</b>) of that space if the union of the collection is the whole space."

"Cut point (Wikipedia Glossary of Topology)"¬"
If <i>X</i> is a connected space with more than one point, then a point <i>x</i> of <i>X</i> is a cut point if the subspace <i>X</i> − {<i>x</i>} is disconnected."
"<a href=""https://en.wikipedia.org/wiki/Dense_set"" title=""Dense set"">Dense set</a> (Wikipedia Glossary of Topology)"¬"
A set is dense if it has nonempty intersection with every nonempty open set. Equivalently, a set is dense if its closure is the whole space."

If (<i>M</i>, <i>d</i>) is a metric space and <i>S</i> is a subset of <i>M</i>, the diameter of <i>S</i> is the <a href=""https://en.wikipedia.org/wiki/Supremum"" class=""mw-redirect"" title=""Supremum"">supremum</a> of the distances <i>d</i>(<i>x</i>, <i>y</i>), where <i>x</i> and <i>y</i> range over <i>S</i>."
"Discrete metric (Wikipedia Glossary of Topology)"¬"
The discrete metric on a set <i>X</i> is the function <i>d</i>&#160;: <i>X</i> × <i>X</i> &#160;→&#160; <b><a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">R</a></b> such that for all <i>x</i>, <i>y</i> in <i>X</i>, <i>d</i>(<i>x</i>, <i>x</i>) = 0 and <i>d</i>(<i>x</i>, <i>y</i>) = 1 if <i>x</i> ≠ <i>y</i>. The discrete metric induces the discrete topology on <i>X</i>."

"<a href=""https://en.wikipedia.org/wiki/Homeomorphism"" title=""Homeomorphism"">Homeomorphism</a> (Wikipedia Glossary of Topology)"¬"
If <i>X</i> and <i>Y</i> are spaces, a <a href=""https://en.wikipedia.org/wiki/Homeomorphism"" title=""Homeomorphism"">homeomorphism</a> from <i>X</i> to <i>Y</i> is a <a href=""https://en.wikipedia.org/wiki/Bijection"" title=""Bijection"">bijective</a> function <i>f</i>&#160;:&#160;<i>X</i>&#160;→&#160;<i>Y</i> such that <i>f</i> and <i>f</i><sup>−1</sup> are continuous. The spaces <i>X</i> and <i>Y</i> are then said to be <b>homeomorphic</b>. From the standpoint of topology, homeomorphic spaces are identical."
"<a href=""https://en.wikipedia.org/wiki/Homogeneous_space"" title=""Homogeneous space"">Homogeneous</a> (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> is <a href=""https://en.wikipedia.org/wiki/Homogeneous_space"" title=""Homogeneous space"">homogeneous</a> if, for every <i>x</i> and <i>y</i> in <i>X</i>, there is a homeomorphism <i>f</i>&#160;: <i>X</i> &#160;→&#160; <i>X</i> such that <i>f</i>(<i>x</i>) = <i>y</i>. Intuitively, the space looks the same at every point. Every <a href=""https://en.wikipedia.org/wiki/Topological_group"" title=""Topological group"">topological group</a> is homogeneous."
"<a href=""https://en.wikipedia.org/wiki/Homotopic"" class=""mw-redirect"" title=""Homotopic"">Homotopic maps</a> (Wikipedia Glossary of Topology)"¬"
Two continuous maps  <i>f</i>, <i>g</i>&#160;: <i>X</i> &#160;→&#160; <i>Y</i> are <a href=""https://en.wikipedia.org/wiki/Homotopic"" class=""mw-redirect"" title=""Homotopic"">homotopic</a> (in <i>Y</i>) if there is a continuous map <i>H</i>&#160;: <i>X</i> × [0, 1] &#160;→&#160; <i>Y</i> such that <i>H</i>(<i>x</i>, 0) = <i>f</i>(<i>x</i>) and <i>H</i>(<i>x</i>, 1) = <i>g</i>(<i>x</i>) for all <i>x</i> in <i>X</i>. Here, <i>X</i> × [0, 1] is given the product topology. The function <i>H</i> is called a <b>homotopy</b> (in <i>Y</i>) between <i>f</i> and <i>g</i>."

The <a href=""https://en.wikipedia.org/wiki/Interior_(topology)"" title=""Interior (topology)"">interior</a> of a set is the largest open set contained in the original set. A point <i>x</i> is an <a href=""https://en.wikipedia.org/wiki/Isolated_point"" title=""Isolated point"">isolated point</a> if the <a href=""https://en.wikipedia.org/wiki/Singleton_(mathematics)"" title=""Singleton (mathematics)"">singleton</a> {<i>x</i>} is open. More generally, if <i>S</i> is a subset of a space <i>X</i>, and if <i>x</i> is a point of <i>S</i>, then <i>x</i> is an isolated point of <i>S</i> if {<i>x</i>} is open in the subspace topology on <i>S</i>."

The metric spaces are then said to be <b>isometrically isomorphic</b>. From the standpoint of metric space theory, isometrically isomorphic spaces are identical."
"Isometry (Wikipedia Glossary of Topology)"¬"
If (<i>M</i><sub>1</sub>, <i>d</i><sub>1</sub>) and (<i>M</i><sub>2</sub>, <i>d</i><sub>2</sub>) are metric spaces, an isometry from <i>M</i><sub>1</sub> to <i>M</i><sub>2</sub> is a function <i>f</i>&#160;: <i>M</i><sub>1</sub> &#160;→&#160; <i>M</i><sub>2</sub> such that <i>d</i><sub>2</sub>(<i>f</i>(<i>x</i>), <i>f</i>(<i>y</i>)) = <i>d</i><sub>1</sub>(<i>x</i>, <i>y</i>) for all <i>x</i>, <i>y</i> in <i>M</i><sub>1</sub>. Every isometry is <a href=""https://en.wikipedia.org/wiki/Injective_function"" title=""Injective function"">injective</a>, although not every isometry is <a href=""https://en.wikipedia.org/wiki/Surjection"" class=""mw-redirect"" title=""Surjection"">surjective</a>."

"<a href=""https://en.wikipedia.org/wiki/Kuratowski_closure_axioms"" title=""Kuratowski closure axioms"">Kuratowski closure axioms</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Kuratowski_closure_axioms"" title=""Kuratowski closure axioms"">Kuratowski closure axioms</a> is a set of <a href=""https://en.wikipedia.org/wiki/Axiom"" title=""Axiom"">axioms</a> satisfied by the function which takes each subset of <i>X</i> to its closure:
<ol align=""left"">
<li><i><a href=""https://en.wikipedia.org/wiki/Isotone_function"" class=""mw-redirect"" title=""Isotone function"">Isotonicity</a></i>: Every set is contained in its closure.</li>
<li><i><a href=""https://en.wikipedia.org/wiki/Idempotent_function"" class=""mw-redirect"" title=""Idempotent function"">Idempotence</a></i>: The closure of the closure of a set is equal to the closure of that set.</li>
<li><i>Preservation of binary unions</i>: The closure of the union of two sets is the union of their closures.</li>
<li><i>Preservation of nullary unions</i>: The closure of the empty set is empty.</li></ol></dd>
<dd>If <i>c</i> is a function from the <a href=""https://en.wikipedia.org/wiki/Power_set"" title=""Power set"">power set</a> of <i>X</i> to itself, then <i>c</i> is a <b>closure operator</b> if it satisfies the Kuratowski closure axioms. The Kuratowski closure axioms can then be used to define a topology on <i>X</i> by declaring the closed sets to be the <a href=""https://en.wikipedia.org/wiki/Fixed_point_(mathematics)"" title=""Fixed point (mathematics)"">fixed points</a> of this operator, i.e. a set <i>A</i> is closed <a href=""https://en.wikipedia.org/wiki/If_and_only_if"" title=""If and only if"">if and only if</a> <i>c</i>(<i>A</i>) = <i>A</i>."

"Locally (P) space (Wikipedia Glossary of Topology)"¬"
There are two definitions for a space to be ""locally (P)"" where (P) is a topological or set-theoretic property: that each point has a neighbourhood with property (P), or that every point has a neighourbood base for which each member has property (P).  The first definition is usually taken for locally compact, countably compact, metrisable, separable, countable; the second for locally connected.<sup id=""cite_ref-EGT65_13-0"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-EGT65-13"">&#91;13&#93;</a></sup>"

If <i>x</i> is a point in a space <i>X</i>, a <a href=""https://en.wikipedia.org/wiki/Loop_(topology)"" title=""Loop (topology)"">loop</a> at <i>x</i> in <i>X</i> (or a loop in <i>X</i> with basepoint <i>x</i>) is a path <i>f</i> in <i>X</i>, such that <i>f</i>(0) = <i>f</i>(1) = <i>x</i>. If <i>A</i> is not meagre in <i>X</i>, <i>A</i> is of <b>second category</b> in <i>X</i>.<sup id=""cite_ref-ss7_14-0"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss7-14"">&#91;14&#93;</a></sup>"

"<a href=""https://en.wikipedia.org/wiki/Metric_map"" title=""Metric map"">Metric map</a> (Wikipedia Glossary of Topology)"¬"
If <i>X</i> and <i>Y</i> are metric spaces with metrics <i>d</i><sub><i>X</i></sub> and <i>d</i><sub><i>Y</i></sub> respectively, then a <a href=""https://en.wikipedia.org/wiki/Metric_map"" title=""Metric map"">metric map</a> is a function <i>f</i> from <i>X</i> to <i>Y</i>, such that for any points <i>x</i> and <i>y</i> in <i>X</i>, <i>d</i><sub><i>Y</i></sub>(<i>f</i>(<i>x</i>), <i>f</i>(<i>y</i>)) ≤ <i>d</i><sub><i>X</i></sub>(<i>x</i>, <i>y</i>). A metric map is <a href=""https://en.wikipedia.org/wiki/Metric_map"" title=""Metric map"">strictly metric</a> if the above inequality is strict for all <i>x</i> and <i>y</i> in <i>X</i>."
"<a href=""https://en.wikipedia.org/wiki/Metric_space"" title=""Metric space"">Metric space</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Metric_space"" title=""Metric space"">metric space</a> (<i>M</i>, <i>d</i>) is a set <i>M</i> equipped with a function <i>d</i>&#160;:&#160;<i>M</i>&#160;×&#160;<i>M</i>&#160;→&#160;<b><a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">R</a></b> satisfying the following axioms for all <i>x</i>, <i>y</i>, and <i>z</i> in <i>M</i>:
<ol align=""left"">
<li><i>d</i>(<i>x</i>, <i>y</i>) ≥ 0</li>
<li><i>d</i>(<i>x</i>, <i>x</i>) = 0</li>
<li>if &#160; <i>d</i>(<i>x</i>, <i>y</i>) = 0 &#160; then &#160; <i>x</i> = <i>y</i> &#160;&#160;&#160; (<i>identity of indiscernibles</i>)</li>
<li><i>d</i>(<i>x</i>, <i>y</i>) = <i>d</i>(<i>y</i>, <i>x</i>)  &#160;&#160;&#160; (<i>symmetry</i>)</li>
<li><i>d</i>(<i>x</i>, <i>z</i>) ≤ <i>d</i>(<i>x</i>, <i>y</i>) + <i>d</i>(<i>y</i>, <i>z</i>) &#160;&#160;&#160;  (<i><a href=""https://en.wikipedia.org/wiki/Triangle_inequality"" title=""Triangle inequality"">triangle inequality</a></i>)</li>
</ol>

<dl><dd>The function <i>d</i> is a <b>metric</b> on <i>M</i>, and <i>d</i>(<i>x</i>, <i>y</i>) is the <b>distance</b> between <i>x</i> and <i>y</i>. The collection of all open balls of <i>M</i> is a base for a topology on <i>M</i>; this is the topology on <i>M</i> induced by <i>d</i>. Every metric space is Hausdorff and paracompact (and hence normal and Tychonoff). Every metric space is first-countable."

A space is <a href=""https://en.wikipedia.org/wiki/Metrizable"" class=""mw-redirect"" title=""Metrizable"">metrizable</a> if it is homeomorphic to a metric space. Every metrizable space is Hausdorff and paracompact (and hence normal and Tychonoff). Every metrizable space is first-countable."

"<a href=""https://en.wikipedia.org/wiki/Neighbourhood_(mathematics)"" title=""Neighbourhood (mathematics)"">Neighbourhood</a><b>/</b>Neighborhood (Wikipedia Glossary of Topology)"¬"
A neighbourhood of a point <i>x</i> is a set containing an open set which in turn contains the point <i>x</i>. More generally, a neighbourhood of a set <i>S</i> is a set containing an open set which in turn contains the set <i>S</i>. A neighbourhood of a point <i>x</i> is thus a neighbourhood of the <a href=""https://en.wikipedia.org/wiki/Singleton_(mathematics)"" title=""Singleton (mathematics)"">singleton</a> set {<i>x</i>}. (Note that under this definition, the neighbourhood itself need not be open. Many authors require that neighbourhoods be open; be careful to note conventions.)"

"<a href=""https://en.wikipedia.org/wiki/Net_(mathematics)"" title=""Net (mathematics)"">Net</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Net_(mathematics)"" title=""Net (mathematics)"">net</a> in a space <i>X</i> is a map from a <a href=""https://en.wikipedia.org/wiki/Directed_set"" title=""Directed set"">directed set</a> <i>A</i> to <i>X</i>. A net from <i>A</i> to <i>X</i> is usually denoted (<i>x</i><sub>α</sub>), where α is an <a href=""https://en.wikipedia.org/wiki/Index_set"" title=""Index set"">index variable</a> ranging over <i>A</i>. Every <a href=""https://en.wikipedia.org/wiki/Sequence"" title=""Sequence"">sequence</a> is a net, taking <i>A</i> to be the directed set of <a href=""https://en.wikipedia.org/wiki/Natural_number"" title=""Natural number"">natural numbers</a> with the usual ordering."
"<a href=""https://en.wikipedia.org/wiki/Normal_space"" title=""Normal space"">Normal</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Normal_space"" title=""Normal space"">normal</a> if any two disjoint closed sets have disjoint neighbourhoods.<sup id=""cite_ref-ss162_6-4"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss162-6"">&#91;6&#93;</a></sup>  Every normal space admits a partition of unity."
"<a href=""https://en.wikipedia.org/wiki/T4_space"" class=""mw-redirect"" title=""T4 space"">Normal Hausdorff</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/T4_space"" class=""mw-redirect"" title=""T4 space"">normal Hausdorff</a> space (or <a href=""https://en.wikipedia.org/wiki/T4_space"" class=""mw-redirect"" title=""T4 space""><b>T<sub>4</sub></b> space</a>) is a normal T<sub>1</sub> space. (A normal space is Hausdorff <a href=""https://en.wikipedia.org/wiki/If_and_only_if"" title=""If and only if"">if and only if</a> it is T<sub>1</sub>, so the terminology is consistent.) Every normal Hausdorff space is Tychonoff."
"<a href=""https://en.wikipedia.org/wiki/Nowhere_dense_set"" title=""Nowhere dense set"">Nowhere dense</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Nowhere_dense_set"" title=""Nowhere dense set"">nowhere dense set</a> is a set whose closure has empty interior."

An <a href=""https://en.wikipedia.org/wiki/Open_cover"" class=""mw-redirect"" title=""Open cover"">open cover</a> is a cover consisting of open sets.<sup id=""cite_ref-ss163_4-3"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss163-4"">&#91;4&#93;</a></sup>"
"Open ball (Wikipedia Glossary of Topology)"¬"
If (<i>M</i>, <i>d</i>) is a metric space, an open ball is a set of the form <i>B</i>(<i>x</i>; <i>r</i>)&#160;:= {<i>y</i> in <i>M</i>&#160;: <i>d</i>(<i>x</i>, <i>y</i>) &lt; <i>r</i>}, where <i>x</i> is in <i>M</i> and <i>r</i> is a <a href=""https://en.wikipedia.org/wiki/Positive_number"" class=""mw-redirect"" title=""Positive number"">positive</a> <a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">real number</a>, the <b>radius</b> of the ball. An open ball of radius <i>r</i> is an <b>open <i>r</i>-ball</b>. Every open ball is an open set in the topology on <i>M</i> induced by <i>d</i>."

"<a href=""https://en.wikipedia.org/wiki/Open_map"" class=""mw-redirect"" title=""Open map"">Open function</a> (Wikipedia Glossary of Topology)"¬"
A function from one space to another is <a href=""https://en.wikipedia.org/wiki/Open_map"" class=""mw-redirect"" title=""Open map"">open</a> if the <a href=""https://en.wikipedia.org/wiki/Image_(mathematics)"" title=""Image (mathematics)"">image</a> of every open set is open."
"Open property (Wikipedia Glossary of Topology)"¬"
A property of points in a <a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">topological space</a> is said to be ""open"" if those points which possess it form an <a href=""https://en.wikipedia.org/wiki/Open_set"" title=""Open set"">open set</a>.  Such conditions often take a common form, and that form can be said to be an <i>open condition</i>; for example, in <a href=""https://en.wikipedia.org/wiki/Metric_space"" title=""Metric space"">metric spaces</a>, one defines an open ball as above, and says that ""strict inequality is an open condition""."
"<a href=""https://en.wikipedia.org/wiki/Paracompact_space"" title=""Paracompact space"">Paracompact</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Paracompact_space"" title=""Paracompact space"">paracompact</a> if every open cover has a locally finite open refinement.  Paracompact implies metacompact.<sup id=""cite_ref-ss23_15-0"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss23-15"">&#91;15&#93;</a></sup>  Paracompact Hausdorff spaces are normal.<sup id=""cite_ref-ss25_16-0"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss25-16"">&#91;16&#93;</a></sup>"
"<a href=""https://en.wikipedia.org/wiki/Partition_of_unity"" title=""Partition of unity"">Partition of unity</a> (Wikipedia Glossary of Topology)"¬"
A partition of unity of a space <i>X</i> is a set of continuous functions from <i>X</i> to [0, 1] such that any point has a neighbourhood where all but a <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> number of the functions are identically zero, and the sum of all the functions on the entire space is identically 1."
"<a href=""https://en.wikipedia.org/wiki/Path_(topology)"" title=""Path (topology)"">Path</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Path_(topology)"" title=""Path (topology)"">path</a> in a space <i>X</i> is a continuous map <i>f</i> from the closed unit <a href=""https://en.wikipedia.org/wiki/Interval_(mathematics)"" title=""Interval (mathematics)"">interval</a> [0, 1] into <i>X</i>. The point <i>f</i>(0) is the initial point of <i>f</i>; the point <i>f</i>(1) is the terminal point of <i>f</i>.<sup id=""cite_ref-ss29_11-2"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss29-11"">&#91;11&#93;</a></sup>"
"<a href=""https://en.wikipedia.org/wiki/Path-connected_space"" class=""mw-redirect"" title=""Path-connected space"">Path-connected</a> (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> is <a href=""https://en.wikipedia.org/wiki/Path-connected_space"" class=""mw-redirect"" title=""Path-connected space"">path-connected</a> if, for every two points <i>x</i>, <i>y</i> in <i>X</i>, there is a path <i>f</i> from <i>x</i> to <i>y</i>, i.e., a path with initial point <i>f</i>(0) = <i>x</i> and terminal point <i>f</i>(1) = <i>y</i>. Every path-connected space is connected.<sup id=""cite_ref-ss29_11-3"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss29-11"">&#91;11&#93;</a></sup>"
"Path-connected component (Wikipedia Glossary of Topology)"¬"
A path-connected component of a space is a maximal nonempty path-connected subspace. The set of path-connected components of a space is a <a href=""https://en.wikipedia.org/wiki/Partition_of_a_set"" title=""Partition of a set"">partition</a> of that space, which is <a href=""https://en.wikipedia.org/wiki/Partition_of_a_set"" title=""Partition of a set"">finer</a> than the partition into connected components.<sup id=""cite_ref-ss29_11-4"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss29-11"">&#91;11&#93;</a></sup> The set of path-connected components of a space <i>X</i> is denoted <a href=""https://en.wikipedia.org/wiki/Homotopy_groups"" class=""mw-redirect"" title=""Homotopy groups"">π<sub>0</sub>(<i>X</i>)</a>."

The prodiscrete topology on a product <i>A</i><sup><i>G</i></sup> is the product topology when each factor <i>A</i> is given the discrete topology.<sup id=""cite_ref-18"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-18"">&#91;18&#93;</a></sup>"

"Pseudocompact (Wikipedia Glossary of Topology)"¬"
A space is pseudocompact if every <a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">real-valued</a> continuous function on the space is bounded."

"Pseudometric space (Wikipedia Glossary of Topology)"¬"
A pseudometric space (<i>M</i>, <i>d</i>) is a set <i>M</i> equipped with a function <i>d</i>&#160;:&#160;<i>M</i>&#160;×&#160;<i>M</i>&#160;→&#160;<b><a href=""https://en.wikipedia.org/wiki/Real_number"" title=""Real number"">R</a></b> satisfying all the conditions of a metric space, except possibly the identity of indiscernibles. That is, points in a pseudometric space may be ""infinitely close"" without being identical.  The function <i>d</i> is a <b>pseudometric</b> on <i>M</i>. Every metric is a pseudometric."
"Punctured neighbourhood<b>/</b>Punctured neighborhood (Wikipedia Glossary of Topology)"¬"
A punctured neighbourhood of a point <i>x</i> is a neighbourhood of <i>x</i>, minus {<i>x</i>}. For instance, the <a href=""https://en.wikipedia.org/wiki/Interval_(mathematics)"" title=""Interval (mathematics)"">interval</a> (−1, 1) = {<i>y</i>&#160;: −1 &lt; <i>y</i> &lt; 1} is a neighbourhood of <i>x</i> = 0 in the <a href=""https://en.wikipedia.org/wiki/Real_line"" title=""Real line"">real line</a>, so the set (−1, 0) ∪ (0, 1) = (−1, 1) − {0} is a punctured neighbourhood of 0."

"<a href=""https://en.wikipedia.org/wiki/Quotient_space_(topology)"" title=""Quotient space (topology)"">Quotient space</a> (Wikipedia Glossary of Topology)"¬"
If <i>X</i> is a space, <i>Y</i> is a set, and <i>f</i>&#160;:&#160;<i>X</i>&#160;→&#160;<i>Y</i> is any <a href=""https://en.wikipedia.org/wiki/Surjection"" class=""mw-redirect"" title=""Surjection"">surjective</a> function, then the <a href=""https://en.wikipedia.org/wiki/Quotient_space_(topology)"" title=""Quotient space (topology)"">quotient topology</a> on <i>Y</i> induced by <i>f</i> is the finest topology for which <i>f</i> is continuous. The space <i>X</i> is a quotient space or <b>identification space</b>. By definition, <i>f</i> is a quotient map. The most common example of this is to consider an <a href=""https://en.wikipedia.org/wiki/Equivalence_relation"" title=""Equivalence relation"">equivalence relation</a> on <i>X</i>, with <i>Y</i> the set of <a href=""https://en.wikipedia.org/wiki/Equivalence_class"" title=""Equivalence class"">equivalence classes</a> and <i>f</i> the natural projection map. This construction is dual to the construction of the subspace topology."
"Refinement (Wikipedia Glossary of Topology)"¬"

"<a href=""https://en.wikipedia.org/wiki/Regular_space"" title=""Regular space"">Regular</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Regular_space"" title=""Regular space"">regular</a> if, whenever <i>C</i> is a closed set and <i>x</i> is a point not in <i>C</i>, then <i>C</i> and <i>x</i> have <a href=""https://en.wikipedia.org/wiki/Disjoint_sets"" title=""Disjoint sets"">disjoint</a> neighbourhoods."
"<a href=""https://en.wikipedia.org/wiki/T3_space"" class=""mw-redirect"" title=""T3 space"">Regular Hausdorff</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/T3_space"" class=""mw-redirect"" title=""T3 space"">regular Hausdorff</a> (or <b>T<sub>3</sub></b>) if it is a regular T<sub>0</sub> space.  (A regular space is Hausdorff <a href=""https://en.wikipedia.org/wiki/If_and_only_if"" title=""If and only if"">if and only if</a> it is T<sub>0</sub>, so the terminology is consistent.)"
"<span id=""Regular_open"">Regular open</span (Wikipedia Glossary of Topology)"¬"
A subset of a space <i>X</i> is regular open if it equals the interior of its closure; dually, a regular closed set is equal to the closure of its interior.<sup id=""cite_ref-Steen_&amp;_Seebach_1978_p.6_19-0"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-Steen_&amp;_Seebach_1978_p.6-19"">&#91;19&#93;</a></sup>  An example of a non-regular open set is the set  <i>U</i> = <span class=""texhtml"">(0,1)</span> ∪ <span class=""texhtml"">(1,2)</span> in <b>R</b> with its normal topology, since 1 is in the interior of the closure of <i>U</i>, but not in <i>U</i>. The regular open subsets of a space form a <a href=""https://en.wikipedia.org/wiki/Complete_Boolean_algebra"" title=""Complete Boolean algebra"">complete Boolean algebra</a>.<sup id=""cite_ref-Steen_&amp;_Seebach_1978_p.6_19-1"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-Steen_&amp;_Seebach_1978_p.6-19"">&#91;19&#93;</a></sup>"
"<a href=""https://en.wikipedia.org/wiki/Relatively_compact"" class=""mw-redirect"" title=""Relatively compact"">Relatively compact</a> (Wikipedia Glossary of Topology)"¬"
A subset <i>Y</i> of a space <i>X</i> is <a href=""https://en.wikipedia.org/wiki/Relatively_compact"" class=""mw-redirect"" title=""Relatively compact"">relatively compact</a> in <i>X</i> if the closure of <i>Y</i> in <i>X</i> is compact."
"Residual (Wikipedia Glossary of Topology)"¬"
If <i>X</i> is a space and <i>A</i> is a subset of <i>X</i>, then <i>A</i> is residual in <i>X</i> if the complement of <i>A</i> is meagre in <i>X</i>. Also called <b>comeagre</b> or <b>comeager</b>."
"Resolvable (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">topological space</a> is called <a href=""https://en.wikipedia.org/wiki/Resolvable_space"" title=""Resolvable space"">resolvable</a> if it is expressible as the union of two <a href=""https://en.wikipedia.org/wiki/Disjoint_sets"" title=""Disjoint sets"">disjoint</a> <a href=""https://en.wikipedia.org/wiki/Dense_subset"" class=""mw-redirect"" title=""Dense subset"">dense subsets</a>."
"Rim-compact (Wikipedia Glossary of Topology)"¬"
A space is rim-compact if it has a base of open sets whose boundaries are compact."
"<a href=""https://en.wikipedia.org/wiki/S_and_L_spaces"" title=""S and L spaces"">S-space</a> (Wikipedia Glossary of Topology)"¬"
An <i>S-space</i> is a <a href=""https://en.wikipedia.org/wiki/Hereditary_property#In_topology"" title=""Hereditary property"">hereditarily</a> <a href=""https://en.wikipedia.org/wiki/Separable_space"" title=""Separable space"">separable</a> space which is not hereditarily <a href=""https://en.wikipedia.org/wiki/Lindel%C3%B6f_space"" title=""Lindelöf space"">Lindelöf</a>.<sup id=""cite_ref-GKW290_12-1"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-GKW290-12"">&#91;12&#93;</a></sup>"
"Scattered (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> is scattered if every nonempty subset <i>A</i> of <i>X</i> contains a point isolated in <i>A</i>."
"<a href=""https://en.wikipedia.org/wiki/Scott_continuity"" title=""Scott continuity"">Scott</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Scott_topology"" class=""mw-redirect"" title=""Scott topology"">Scott topology</a> on a <a href=""https://en.wikipedia.org/wiki/Poset"" class=""mw-redirect"" title=""Poset"">poset</a> is that in which the open sets are those <a href=""https://en.wikipedia.org/wiki/Upper_set"" title=""Upper set"">Upper sets</a> inaccessible by directed joins.<sup id=""cite_ref-20"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-20"">&#91;20&#93;</a></sup>"
"Second category (Wikipedia Glossary of Topology)"¬"
See <b>Meagre</b>."
"<a href=""https://en.wikipedia.org/wiki/Second-countable_space"" title=""Second-countable space"">Second-countable</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Second-countable_space"" title=""Second-countable space"">second-countable</a> or <b>perfectly separable</b> if it has a <a href=""https://en.wikipedia.org/wiki/Countable"" class=""mw-redirect"" title=""Countable"">countable</a> base for its topology.<sup id=""cite_ref-ss162_6-6"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss162-6"">&#91;6&#93;</a></sup> Every second-countable space is first-countable, separable, and Lindelöf."
"<a href=""https://en.wikipedia.org/wiki/Semilocally_simply_connected"" class=""mw-redirect"" title=""Semilocally simply connected"">Semilocally simply connected</a> (Wikipedia Glossary of Topology)"¬"
A space <i>X</i> is <a href=""https://en.wikipedia.org/wiki/Semilocally_simply_connected"" class=""mw-redirect"" title=""Semilocally simply connected"">semilocally simply connected</a> if, for every point <i>x</i> in <i>X</i>, there is a neighbourhood <i>U</i> of <i>x</i> such that every loop at <i>x</i> in <i>U</i> is homotopic in <i>X</i> to the constant loop <i>x</i>. Every simply connected space and every locally simply connected space is semilocally simply connected. (Compare with locally simply connected; here, the homotopy is allowed to live in <i>X</i>, whereas in the definition of locally simply connected, the homotopy must live in <i>U</i>.)"
"<a href=""https://en.wikipedia.org/wiki/Semiregular_space"" title=""Semiregular space"">Semiregular</a> (Wikipedia Glossary of Topology)"¬"
A space is semiregular if the regular open sets form a base."
"<a href=""https://en.wikipedia.org/wiki/Separable_(topology)"" class=""mw-redirect"" title=""Separable (topology)"">Separable</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Separable_(topology)"" class=""mw-redirect"" title=""Separable (topology)"">separable</a> if it has a <a href=""https://en.wikipedia.org/wiki/Countable"" class=""mw-redirect"" title=""Countable"">countable</a> dense subset.<sup id=""cite_ref-ss162_6-7"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss162-6"">&#91;6&#93;</a></sup><sup id=""cite_ref-ss7_14-1"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss7-14"">&#91;14&#93;</a></sup>"
"<a href=""https://en.wikipedia.org/wiki/Separated_sets"" title=""Separated sets"">Separated</a> (Wikipedia Glossary of Topology)"¬"
Two sets <i>A</i> and <i>B</i> are <a href=""https://en.wikipedia.org/wiki/Separated_sets"" title=""Separated sets"">separated</a> if each is <a href=""https://en.wikipedia.org/wiki/Disjoint_sets"" title=""Disjoint sets"">disjoint</a> from the other's closure."
"Sequentially compact (Wikipedia Glossary of Topology)"¬"
A space is sequentially compact if every <a href=""https://en.wikipedia.org/wiki/Sequence"" title=""Sequence"">sequence</a> has a convergent subsequence. Every sequentially compact space is countably compact, and every first-countable, countably compact space is sequentially compact."

See <b><a href=""https://en.wikipedia.org/wiki/Metric_map"" title=""Metric map"">metric map</a></b>"
"<a href=""https://en.wikipedia.org/wiki/Simply_connected_space"" title=""Simply connected space"">Simply connected</a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/Simply_connected_space"" title=""Simply connected space"">simply connected</a> if it is path-connected and every loop is homotopic to a constant map."

"<a href=""https://en.wikipedia.org/wiki/Sober_space"" title=""Sober space"">Sober</a> (Wikipedia Glossary of Topology)"¬"
In a <a href=""https://en.wikipedia.org/wiki/Sober_space"" title=""Sober space"">sober space</a>, every <a href=""https://en.wikipedia.org/wiki/Hyperconnected_space"" title=""Hyperconnected space"">irreducible</a> closed subset is the <a href=""https://en.wikipedia.org/wiki/Closure_(topology)"" title=""Closure (topology)"">closure</a> of exactly one point: that is, has a unique <a href=""https://en.wikipedia.org/wiki/Generic_point"" title=""Generic point"">generic point</a>.<sup id=""cite_ref-21"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-21"">&#91;21&#93;</a></sup>"
"Star (Wikipedia Glossary of Topology)"¬"
The star of a point in a given <a href=""https://en.wikipedia.org/wiki/Cover_(topology)"" title=""Cover (topology)"">cover</a> of a <a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">topological space</a> is the union of all the sets in the cover that contain the point. See <b><a href=""https://en.wikipedia.org/wiki/Star_refinement"" title=""Star refinement"">star refinement</a></b>."

"<a href=""https://en.wikipedia.org/wiki/Subbase"" title=""Subbase"">Subbase</a> (Wikipedia Glossary of Topology)"¬"
A collection of open sets is a <a href=""https://en.wikipedia.org/wiki/Subbase"" title=""Subbase"">subbase</a> (or <b>subbasis</b>) for a topology if every non-empty proper open set in the topology is a union of <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> intersections of sets in the subbase. If <i>B</i> is <i>any</i> collection of subsets of a set <i>X</i>, the topology on <i>X</i> generated by <i>B</i> is the smallest topology containing <i>B</i>; this topology consists of the empty set, <i>X</i> and all unions of finite intersections of elements of <i>B</i>."

"Submaximal space (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">topological space</a> is said to be <b>submaximal</b> if every subset of it is locally closed, that is, every subset is the intersection of an <a href=""https://en.wikipedia.org/wiki/Open_set"" title=""Open set"">open set</a> and a <a href=""https://en.wikipedia.org/wiki/Closed_set"" title=""Closed set"">closed set</a>.</dd></dl>
<p>Here are some facts about submaximality as a property of topological spaces:
</p>
<ul align=""left"">
<li>Every <a href=""https://en.wikipedia.org/wiki/Door_space"" title=""Door space"">door space</a> is submaximal.</li>
<li>Every submaximal space is <i>weakly submaximal</i> viz every finite set is locally closed.</li>
<li>Every submaximal space is <a href=""https://en.wikipedia.org/wiki/Irresolvable_space"" class=""mw-redirect"" title=""Irresolvable space"">irresolvable</a><sup id=""cite_ref-22"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-22"">&#91;22&#93;</a></sup></li></ul>"
"Subspace (Wikipedia Glossary of Topology)"¬"
If <i>T</i> is a topology on a space <i>X</i>, and if <i>A</i> is a subset of <i>X</i>, then the <a href=""https://en.wikipedia.org/wiki/Subspace_topology"" title=""Subspace topology"">subspace topology</a> on <i>A</i> induced by <i>T</i> consists of all intersections of open sets in <i>T</i> with <i>A</i>. This construction is dual to the construction of the quotient topology."
"<a href=""https://en.wikipedia.org/wiki/T0_space"" class=""mw-redirect"" title=""T0 space"">T<sub>0</sub></a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/T0_space"" class=""mw-redirect"" title=""T0 space"">T<sub>0</sub></a> (or <b>Kolmogorov</b>) if for every pair of distinct points <i>x</i> and <i>y</i> in the space, either there is an open set containing <i>x</i> but not <i>y</i>, or there is an open set containing <i>y</i> but not <i>x</i>."
"<a href=""https://en.wikipedia.org/wiki/T1_space"" title=""T1 space"">T<sub>1</sub></a> (Wikipedia Glossary of Topology)"¬"
A space is <a href=""https://en.wikipedia.org/wiki/T1_space"" title=""T1 space"">T<sub>1</sub></a> (or <b>Fréchet</b> or <b>accessible</b>) if for every pair of distinct points <i>x</i> and <i>y</i> in the space, there is an open set containing <i>x</i> but not <i>y</i>. (Compare with T<sub>0</sub>; here, we are allowed to specify which point will be contained in the open set.) Equivalently, a space is T<sub>1</sub> if all its <a href=""https://en.wikipedia.org/wiki/Singleton_(mathematics)"" title=""Singleton (mathematics)"">singletons</a> are closed. Every T<sub>1</sub> space is T<sub>0</sub>."

"<a href=""https://en.wikipedia.org/wiki/Topological_invariant"" class=""mw-redirect"" title=""Topological invariant"">Topological invariant</a> (Wikipedia Glossary of Topology)"¬"
A topological invariant is a property which is preserved under homeomorphism. For example, compactness and connectedness are topological properties, whereas boundedness and completeness are not. <a href=""https://en.wikipedia.org/wiki/Algebraic_topology"" title=""Algebraic topology"">Algebraic topology</a> is the study of topologically invariant <a href=""https://en.wikipedia.org/wiki/Abstract_algebra"" title=""Abstract algebra"">abstract algebra</a> constructions on topological spaces."
"<a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">Topological space</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Topological_space"" title=""Topological space"">topological space</a> (<i>X</i>, <i>T</i>) is a set <i>X</i> equipped with a collection <i>T</i> of subsets of <i>X</i> satisfying the following <a href=""https://en.wikipedia.org/wiki/Axiom"" title=""Axiom"">axioms</a>:</dd></dl>
<dl><dd
<ol align=""left"">
<li>The empty set and <i>X</i> are in <i>T</i>.</li>
<li>The union of any collection of sets in <i>T</i> is also in <i>T</i>.</li>
<li>The intersection of any pair of sets in <i>T</i> is also in <i>T</i>.</li>
</ol>
</dd>
</dl>
<dl><dd>The collection <i>T</i> is a <b>topology</b> on <i>X</i>."

"Topologically complete (Wikipedia Glossary of Topology)"¬"
<a href=""https://en.wikipedia.org/wiki/Completely_metrizable_space"" title=""Completely metrizable space"">Completely metrizable spaces</a> (i. e. topological spaces homeomorphic to complete metric spaces) are often called <i>topologically complete</i>; sometimes the term is also used for <a href=""https://en.wikipedia.org/w/index.php?title=%C4%8Cech-complete_space&amp;action=edit&amp;redlink=1"" class=""new"" title=""Čech-complete space (page does not exist)"">Čech-complete spaces</a> or <a href=""https://en.wikipedia.org/wiki/Completely_uniformizable_space"" title=""Completely uniformizable space"">completely uniformizable spaces</a>."

"Totally bounded (Wikipedia Glossary of Topology)"¬"
A metric space <i>M</i> is totally bounded if, for every <i>r</i> &gt; 0, there exist a <a href=""https://en.wikipedia.org/wiki/Finite_set"" title=""Finite set"">finite</a> cover of <i>M</i> by open balls of radius <i>r</i>. A metric space is compact if and only if it is complete and totally bounded."
"Totally disconnected (Wikipedia Glossary of Topology)"¬"
A space is totally disconnected if it has no connected subset with more than one point."
"<a href=""https://en.wikipedia.org/wiki/Trivial_topology"" title=""Trivial topology"">Trivial topology</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Trivial_topology"" title=""Trivial topology"">trivial topology</a> (or <b>indiscrete topology</b>) on a set <i>X</i> consists of precisely the empty set and the entire space <i>X</i>."
"<a href=""https://en.wikipedia.org/wiki/Tychonoff_space"" title=""Tychonoff space"">Tychonoff</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Tychonoff_space"" title=""Tychonoff space"">Tychonoff space</a> (or <b>completely regular Hausdorff</b> space, <b>completely T<sub>3</sub></b> space, <b>T<sub>3.5</sub></b> space) is a completely regular T<sub>0</sub> space.  (A completely regular space is Hausdorff <a href=""https://en.wikipedia.org/wiki/If_and_only_if"" title=""If and only if"">if and only if</a> it is T<sub>0</sub>, so the terminology is consistent.) Every Tychonoff space is regular Hausdorff."
"Ultra-connected (Wikipedia Glossary of Topology)"¬"
A space is ultra-connected if no two non-empty closed sets are disjoint.<sup id=""cite_ref-ss29_11-5"" class=""reference""><a href=""https://en.wikipedia.org/wiki/Glossary_of_topology#cite_note-ss29-11"">&#91;11&#93;</a></sup> Every ultra-connected space is path-connected."
"<a href=""https://en.wikipedia.org/wiki/Ultrametric_space"" title=""Ultrametric space"">Ultrametric</a> (Wikipedia Glossary of Topology)"¬"
A metric is an ultrametric if it satisfies the following stronger version of the <a href=""https://en.wikipedia.org/wiki/Triangle_inequality"" title=""Triangle inequality"">triangle inequality</a>: for all <i>x</i>, <i>y</i>, <i>z</i> in <i>M</i>, <i>d</i>(<i>x</i>, <i>z</i>) ≤ max(<i>d</i>(<i>x</i>, <i>y</i>), <i>d</i>(<i>y</i>, <i>z</i>))."
"<a href=""https://en.wikipedia.org/wiki/Uniform_isomorphism"" title=""Uniform isomorphism"">Uniform isomorphism</a> (Wikipedia Glossary of Topology)"¬"
If <i>X</i> and <i>Y</i> are <a href=""https://en.wikipedia.org/wiki/Uniform_space"" title=""Uniform space"">uniform spaces</a>, a uniform isomorphism from <i>X</i> to <i>Y</i> is a bijective function <i>f</i>&#160;: <i>X</i> → <i>Y</i> such that <i>f</i> and <i>f</i><sup>−1</sup> are <a href=""https://en.wikipedia.org/wiki/Uniformly_continuous"" class=""mw-redirect"" title=""Uniformly continuous"">uniformly continuous</a>. The spaces are then said to be uniformly isomorphic and share the same <a href=""https://en.wikipedia.org/wiki/Uniform_properties"" class=""mw-redirect"" title=""Uniform properties"">uniform properties</a>."
"<a href=""https://en.wikipedia.org/wiki/Uniformizable"" class=""mw-redirect"" title=""Uniformizable"">Uniformizable</a>/Uniformisable (Wikipedia Glossary of Topology)"¬"
A space is uniformizable if it is homeomorphic to a uniform space."
"<a href=""https://en.wikipedia.org/wiki/Uniform_space"" title=""Uniform space"">Uniform space</a> (Wikipedia Glossary of Topology)"¬"
A <a href=""https://en.wikipedia.org/wiki/Uniform_space"" title=""Uniform space"">uniform space</a> is a set <i>X</i> equipped with a nonempty collection Φ of subsets of the <a href=""https://en.wikipedia.org/wiki/Cartesian_product"" title=""Cartesian product"">Cartesian product</a> <i>X</i> × <i>X</i> satisfying the following <a href=""https://en.wikipedia.org/wiki/Axiom"" title=""Axiom"">axioms</a>:</dd></dl>
<dl><dd>
<ol align=""left"">
<li>if <i>U</i> is in Φ, then <i>U</i> contains { (<i>x</i>, <i>x</i>) | <i>x</i> in <i>X</i> }.</li>
<li>if <i>U</i> is in Φ, then { (<i>y</i>, <i>x</i>) | (<i>x</i>, <i>y</i>) in <i>U</i> } is also in Φ</li>
<li>if <i>U</i> is in Φ and <i>V</i> is a subset of <i>X</i> × <i>X</i> which contains <i>U</i>, then <i>V</i> is in Φ</li>
<li>if <i>U</i> and <i>V</i> are in Φ, then <i>U</i> ∩ <i>V</i> is in Φ</li>
<li>if <i>U</i> is in Φ, then there exists <i>V</i> in Φ such that, whenever (<i>x</i>, <i>y</i>) and (<i>y</i>, <i>z</i>) are in <i>V</i>, then (<i>x</i>, <i>z</i>) is in <i>U</i>.</li>
</ol>
</dd>
</dl>
<dl><dd>The elements of Φ are called <b>entourages</b>, and Φ itself is called a <b>uniform structure</b> on <i>X</i>. The uniform structure induces a topology on <i>X</i> where the basic neighborhoods of <i>x</i> are sets of the form {<i>y</i>&#160;: (<i>x</i>,<i>y</i>)∈<i>U</i>} for <i>U</i>∈Φ."

"<a href=""https://en.wikipedia.org/wiki/Weak_topology"" title=""Weak topology"">Weak topology</a> (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Weak_topology"" title=""Weak topology"">weak topology</a> on a set, with respect to a collection of functions from that set into topological spaces, is the coarsest topology on the set which makes all the functions continuous."

"Weakly countably compact (Wikipedia Glossary of Topology)"¬"
A space is weakly countably compact (or <b>limit point compact</b>) if every <a href=""https://en.wikipedia.org/wiki/Infinity"" title=""Infinity"">infinite</a> subset has a limit point."
"Weakly hereditary (Wikipedia Glossary of Topology)"¬"
A property of spaces is said to be weakly hereditary if whenever a space has that property, then so does every closed subspace of it. For example, compactness and the Lindelöf property are both weakly hereditary properties, although neither is hereditary."
"Weight (Wikipedia Glossary of Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Weight_of_a_space"" class=""mw-redirect"" title=""Weight of a space"">weight of a space</a> <i>X</i> is the smallest <a href=""https://en.wikipedia.org/wiki/Cardinal_number"" title=""Cardinal number"">cardinal number</a> κ such that <i>X</i> has a base of cardinal κ. (Note that such a cardinal number exists, because the entire topology forms a base, and because the class of cardinal numbers is <a href=""https://en.wikipedia.org/wiki/Well_order"" class=""mw-redirect"" title=""Well order"">well-ordered</a>.)"
